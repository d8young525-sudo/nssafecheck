=== Firebase Hosting 배포 가이드 (CLI 방식) ===

전제조건:
1. Firebase Console에서 새 사이트 생성 완료
   사이트 ID: nssafe-inspection (예시)
   생성된 URL: https://nssafe-inspection.web.app

2. 로컬에 Firebase CLI 설치됨
   npm install -g firebase-tools

배포 명령어:
-----------

# 1. Firebase 로그인 (최초 1회)
firebase login

# 2. 프로젝트 폴더로 이동
cd [다운로드한 폴더 경로]

# 3. 타겟 설정 (새 사이트에 배포하기 위해)
firebase target:apply hosting nssafe-inspection nssafe-inspection

# 4. 배포 실행
firebase deploy --only hosting:nssafe-inspection

# 또는 기본 사이트에 배포하려면 (주의!)
firebase deploy --only hosting

배포 완료 후:
- 자동으로 배포 URL이 표시됩니다
- 브라우저에서 https://nssafe-inspection.web.app 접속

주의사항:
- 반드시 새 사이트 ID를 확인하세요
- 기존 차량재고현황 사이트와 다른 사이트여야 합니다
